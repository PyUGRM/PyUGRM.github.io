Neben Unicode in Python 3 und den Top 10 Fehlern beim Programmieren in Python, gibt es auch einen Überblick über Nikola und wie damit statische Webseiten erzeugt werden und anschließend via Github Pages unter einer eigenen Domain ausgeliefert werden können.

Das ist besonders interessant für alle, die ohnehin mit Github arbeiten oder Projekte betreuen, die gerne eine eigene Webseite haben möchten, aber die Pflege eines CMS + Datenbank und Webserver scheuen.

pyugrm.de ist live

PyUGRM Orga Team

2014-05-01

Durch die technischen Probleme mit dem Wiki des Python-Forums (mehr dazu hier),
sahen wir uns gezwungen zeitnah einen Ersatz zu schaffen, der uns nicht einfach
so abhanden kommen kann ;-)

Wir haben deshalb die Domain pyugrm.de registriert und ein
Nikola-Blog aufgesetzt, das wir über Github managen können.

Das war das 12. Treffen der PyUGRM
